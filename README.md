# Restaurant Menu Android Application

## 📌 Overview
A mobile restaurant menu application developed using Android Studio that allows users to browse food items through a clean and user-friendly interface.

---

## ⚙️ Features
- Browse restaurant menu
- Food categories
- Interactive mobile UI
- Simple and user-friendly design

---

## 🛠️ Technologies Used
- Android Studio
- Java
- XML
