# 🍔 Tap 'N Eat – Food Delivery Mobile Application

## 📌 Overview
Tap 'N Eat is a dynamic food delivery mobile application developed using Android Studio. The app is designed to simplify the food ordering process by connecting users with local restaurants and delivery services, providing a smooth and convenient user experience.

---

## 🎯 Purpose
The main objective of this application is to streamline food ordering and delivery, making it easier for users to browse restaurants, select meals, and place orders efficiently. It enhances accessibility to a variety of food options through a single mobile platform.

---

## 🛠️ Technologies & Tools Used

- **Android Studio** – Primary IDE used for Android application development  
- **Java** – Core programming language used for application logic  
- **XML** – Used for designing user interfaces and layouts  
- **Figma** – Used for wireframing and UI/UX design during the planning phase  
- **Firebase Realtime Database** – Used to store and manage dynamic data such as user details, restaurant information, and orders  
- **Firebase Authentication** – Used to implement secure user login and authentication system  

---

## 🔥 Backend Integration

The application uses Firebase as its backend service:

- Real-time data synchronization using Firebase Realtime Database  
- Secure authentication for users  
- Cloud-based storage for dynamic application data  

---

## 📱 Project Highlights

- User-friendly food ordering interface  
- Real-time database integration  
- Secure authentication system  
- Clean and intuitive UI design  
- Scalable backend using Firebase
